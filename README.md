# Ahmad Alsmadi — Personal Portfolio

A personal portfolio website that showcases my projects, skills, and professional background as a Full Stack Developer. Built entirely with vanilla HTML, CSS, and JavaScript — no frameworks, libraries, or build tools — to demonstrate a solid understanding of core web fundamentals. The site includes a fully responsive layout, dark/light theme support, smooth scroll-triggered animations, and a working contact form that delivers messages directly to my email.

## Features
- Dark/light theme toggle (persisted via localStorage)
- Typing animation hero
- Scroll-triggered reveal animations
- Dynamic project filtering
- Animated stats counters
- Contact form with real email delivery (Formspree)
- Fully responsive (mobile / tablet / desktop)
- Accessible (keyboard navigation, ARIA labels, WCAG AA contrast)

## Tech Stack
- HTML5
- CSS3 (custom properties, Flexbox/Grid)
- Vanilla JavaScript (Intersection Observer API, Fetch API)

## Sections
Hero · About · Skills · Projects · Experience · Contact
